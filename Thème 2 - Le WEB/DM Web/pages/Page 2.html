La Tesla Model S

La Tesla Model S est une voiture dites 'Milieu de gamme' chez Tesla.
La Tesla Model S même si dite 'Milieu de gamme' a un prix de départ de tout meme 104 990€.
Dans le commun des mortels ce prix peut paraitre éxecif mais ce prix n'est rien compare au voiture thermique actuelle vendue a ce prix.
En effet dans la Tesla Model S et d'ailleurs dans toute les Tesla nous avons l'Autopilot de base integre, permettant d'activer le regulateur de vitesse et de pouvoir lever le pied sans que la voiture freine.
Des options pour l'Autopilot sont possible d'etre achete même après l'achat de la voiture.
Les deux options respectives sont achetable a respectivement 3 800€ et 7 500€ peut importe la Tesla achete (pour le moment)
